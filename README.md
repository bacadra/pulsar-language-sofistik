SOFiSTiK grammar is provide for files with extensions `.dat`, `.gra`, `.grb` and `.results`.

SOFiSTiK RAW grammar is provide for files with extensions `.erg`, `.lst`, `.prt` and `.urs`.

## Installation

### Atom Text Editor

The official Atom packages store has been [disabled](https://github.blog/2022-06-08-sunsetting-atom/). To get latest version run the shell command

    apm install bacadra/atom-language-sofistik

and obtain the package directly from Github repository.

### Pulsar Text Editor

The package has compability with [Pulsar](https://pulsar-edit.dev/) and can be install

    ppm install bacadra/atom-language-sofistik

or directly [language-sofistik](https://web.pulsar-edit.dev/packages/language-sofistik) from Pulsar package store.

## Compatibility

**Support versions of SOFiSTiK are 2023, 2022, 2020 and 2018. English only.**

As the package supports many versions of SOFiSTiK and commands are constantly changing, the package offers hints that may no longer be valid in given version.
